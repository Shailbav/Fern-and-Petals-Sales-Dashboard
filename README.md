Ferns N Petals Sales Dashboard | Power BI
Project Overview:
This Power BI dashboard analyzes sales performance data for Ferns N Petals (FNP), providing actionable insights into revenue, customer purchasing behavior, and product performance. The dashboard helps stakeholders monitor key business metrics and identify growth opportunities through interactive visualizations.


Key Features:
Sales performance monitoring through KPI cards
Revenue and order analysis
Product category performance tracking
Customer purchasing insights
Sales funnel visualization
Interactive tables and charts for detailed analysis
Data-driven decision support for business stakeholders


Tools & Technologies:
Power BI
Power Query
DAX
Data Modeling
Data Visualization


Business Insights:
Identified top-performing products and categories
Analyzed customer purchasing trends
Tracked sales performance across different segments
Monitored key revenue indicators and business KPIs


Dashboard Components:
KPI Cards
Funnel Chart
Donut Chart
Column Chart
Interactive Tables
Filters and Slicers


Outcome:
Developed an interactive business intelligence dashboard that transforms raw sales data into meaningful insights, enabling better decision-making and performance tracking.
